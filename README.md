# Canvas-Heart-Equation
A vibrant, synchronized heartbeat animation built using the **HTML Canvas API**. This project features a parametric heart shape that pulses in time with a shifting rainbow background and "echoes" outward.

## ✨ Features
* **Mathematical Heart Shape**: Drawn using parametric equations.
* **Dynamic Pulse**: Animated using sine waves for a lifelike heartbeat.
* **Rainbow Echoes**: Visual "ghosts" of the heart that fade as they expand.
* **HSL Color Cycling**: A smooth, shifting rainbow background.

## 🚀 How to Run
1.  Clone this repository:
    ```bash
    git clone [https://github.com/YOUR_USERNAME/canvas-heart.git](https://github.com/YOUR_USERNAME/canvas-heart.git)
    ```
2.  Navigate to the folder:
    ```bash
    cd "Canvas Heart Equation"
    ```
3.  Open `index.html` in any modern web browser.

## 🛠️ Built With
* **HTML Canvas** - For rendering the graphics.
* **JavaScript** - For the animation logic and math.
* **CSS** - For the full-screen layout.
